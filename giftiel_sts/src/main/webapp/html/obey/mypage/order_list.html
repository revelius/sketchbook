주문/배송 절차안내

STEP1 결제대기 결제이전의 입금확인중인 상태입니다. 24시간 내 주문취소가 가능하며, 24시간 내 미입금 시 자동취소 됩니다.

STEP2 결제완료 결제가 확인된 상태이며 24시간 내 결제취소가 가능합니다.

STEP3 배송준비 주문한 상품의 배송을 준비하는 과정입니다. * 출고지연 배송준비가 길어지거나 품절상태로 인해 배송이 지연되는 상태입니다.

STEP4 배송중 상품이 배송되어 배송중 인 상태입니다.

STEP5 배송완료 상품이 전달되어 배송이 완료된 상태이며, 7일 내 교환/반품신청이 가능합니다. 배송완료 3일 후 포인트가 적립됩니다.
